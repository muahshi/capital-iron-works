# Capital Iron Works Website (Bhopal)
This is the official, fully responsive single-page marketing website for **Capital Iron Works (Bhopal)**.
Designed in alignment with the brand identity: Bold Orange/Amber, Premium Industrial Dark Black, and Metallic Steel Gray, this portal features smooth-scroll transition animations, seasonal toggles, and direct customer conversion channels.
## 🛠️ Key Features
 * **Modern Responsive Design:** Crafted with Tailwind CSS, adapting smoothly across Mobile, Tablet, and Desktop displays.
 * **Dynamic Seasonal Selector:** Displays Cooler Bodies & Stands in Summer (Apr-Jun), Designer Plant Stands & Accessories in Monsoon/Winter (Jul-Jan), and Slotted Angle Racks for All Year round.
 * **Interactive Price Calculator:** Allows users to pick product types, dimensions, and coating details to see a real-time price estimation.
 * **Conversion Optimization:** Every CTA acts as a direct link, opening WhatsApp with customized context messages pointing to user configurations.
## 🚀 Quick Deployment to GitHub Pages
 1. Create a public repository on GitHub named capital-iron-works.
 2. Upload the index.html file to this repository.
 3. Go to repository **Settings** -> **Pages** (under Code and automation sidebar).
 4. Set the Source to **Deploy from a branch**, select the **main/master** branch, and choose the /root folder. Click Save.
 5. Your professional website will be live at https://<your-username>.github.io/capital-iron-works/ within 2 minutes!
